---
title: "Quick start"
description: "Generate your first alpha pattern in under a minute"
---

<Steps>
  <Step title="Open the app">
    Navigate to the app in a modern browser — Chrome, Firefox, Edge, and Safari are all supported. No installation and no account are required.
  </Step>

  <Step title="Upload an image">
    Drag a JPG, PNG, or WEBP image onto the drop zone, or click the zone to open a file browser. The tool reads the file locally using the browser's `FileReader` API — nothing is sent to a server.

    <Note>
      Transparent PNG images work especially well. The alpha threshold control can cleanly remove semi-transparent border pixels that would otherwise create unwanted fringe colors around the edges of your pattern.
    </Note>
  </Step>

  <Step title="Configure the grid">
    The grid defaults to **50 columns × 50 rows** with aspect-ratio lock enabled.

    Adjust the **Columns** field and the **Rows** field updates automatically to preserve your image's proportions. For a friendship bracelet pattern, a width of 30–80 columns is typical. Wider grids produce more detail; narrower grids produce simpler, faster patterns.
  </Step>

  <Step title="Adjust color filters (optional)">
    Two sliders let you control how colors are handled before the pattern is built:

    - **Color similarity threshold** (0–50) — merges colors within the given RGB distance. Start at `0` and increase gradually until the palette reaches a manageable size for your craft.
    - **Alpha threshold** (0–100%) — removes pixels below the given opacity level. A value of 20–30% is a good starting point for PNG images with soft or anti-aliased edges.
  </Step>

  <Step title="Generate the pattern">
    Click the **Generar Patrón** button. Processing runs in a background Web Worker — a spinner appears while the image is being analyzed, and the canvas updates as soon as the result is ready.
  </Step>

  <Step title="Preview the result">
    The canvas on the right shows the full color grid. Use these controls to inspect it:

    - **Scroll** to zoom in and out.
    - **Drag** to pan across the canvas.
    - **Pencil tool** — switch to the pencil to repaint individual cells by clicking or dragging.
    - **Undo / redo** buttons let you step back through any manual edits.

    The palette legend below the canvas maps each letter symbol (A, B, C, AA…) to its HEX value and RGB components.
  </Step>

  <Step title="Export">
    Choose a format from the export panel and click the corresponding button to download:

    | Format | Best for |
    |--------|----------|
    | PNG | Sharing or printing the pattern as an image |
    | Excel (.xlsx) | Editing in spreadsheet software without color fill |
    | Excel colored (.xls) | Spreadsheet with cells filled in the correct colors |
    | CSV | Importing into other tools or scripts |
    | Google Sheets (.xlsx colored) | Uploading to Google Sheets with color fill |
    | HTML colored | Self-contained HTML page viewable in any browser |

    See the [Export guide](/guides/export) for details on each format.
  </Step>
</Steps>

<Tip>
  Start with a higher color threshold to get a simpler pattern with fewer colors — this is easier to stitch or bead and quicker to count. Once you're happy with the overall shape, reduce the threshold and regenerate to add more color detail.
</Tip>
