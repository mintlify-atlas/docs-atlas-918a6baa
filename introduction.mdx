Image to Alpha Pattern is a browser-based tool that converts any photo or graphic into a **symbol-labeled color matrix** — the kind used for alpha patterns in friendship bracelet making, cross-stitch, perler bead art, and pixel art crafts. No installation, no account, no server uploads: everything runs locally in your browser.

## How the pattern works

When you generate a pattern, the tool divides your image into an N×M grid. For each cell it samples the average color of that region of the image. Colors that are sufficiently similar (within a configurable RGB-distance threshold) are merged into a single palette entry, keeping the final pattern practical to stitch or bead.

Every unique color in the resulting palette is assigned a letter symbol — A, B, C, and so on, extending to two-character sequences (AA, AB, AC…) when the palette grows large. The output is two things:

- **Symbol matrix** — a grid of letter codes, one per cell, that tells you which color goes where.
- **Palette legend** — a table mapping each symbol to its exact HEX value and RGB components.

Together these give you everything you need to reproduce the image stitch-by-stitch or bead-by-bead.

## Workflow overview

The typical workflow takes less than a minute:

1. **Upload** a JPG, PNG, or WEBP image.
2. **Configure** the grid dimensions and color filters.
3. **Generate** the pattern with a single button click.
4. **Preview and edit** the result on the interactive canvas.
5. **Export** in the format that fits your craft workflow.
